# 12-Week Roadmap

### Phase 1 (Weeks 1–4) — MVP
- Vite/React/Tailwind scaffold
- PWA manifest + offline caching
- PDF ingest/viewer
- Regex detectors (SSN, phone, email, creditCard, DOB)
- Burn-in export + download

### Phase 2 (Weeks 5–8) — UX & Trust
- Wizard: select PII categories + preview matches
- Dexie storage, undo history
- Privacy explainers
- Stripe paywall (free 1 file/day)

### Phase 3 (Weeks 9–12) — Smart Detection
- Names via `compromise`
- Custom regex library
- Batch processing
- OCR validation tests
